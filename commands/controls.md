# Controls
## Primary Controller
### Drive System
- Move left/right - Left Stick X-axis
- Move forward/backward - Left Stick Y-axis
- Rotate - Right Stick X-axis
- Brace (wheels in x-pattern) - Left Stick Press + Right Stick Press
- Re-center gyro - Minus + Right Stick Press
- Toggle field relative drive (default: enabled) - Minus + Plus

### Intake
- Toggle position - X
- Enable intake motor - (with intake in low position) Left Trigger (hold)
- Reverse intake motor - (with intake in low position) Left Trigger (hold) + D-pad Left (hold)

### *DEBUG (not for match use)*
- Toggle Feeder - Minus + Y
- Toggle Shooter - Minus + B

## Secondary Controller
### Feeder
- Enable feeder intake - Right Bumper (hold)
- Reverse feeder intake - Right Bumper (hold) + D-pad Left (hold)

### Shooter
- Shoot sequence - Right Trigger (hold)
- Toggle hub targeting - A
  - cancels backwards aim, if active
- Toggle backwards aim - X
  - cancels hub targeting, if active
- Manually move turret - Left Bumper (hold) + Right Stick X-axis
- Manually tilt shooter - Left Bumper (hold) + Right Stick Y-axis
  - resets to lowest point when Left Bumper released
